### 3. Интеграция (5 мин)
```bash
# Pi-hole
scp dynamic-blocklist.txt pi@pihole:/etc/pihole/

# dnsmasq
sudo cp dynamic-blocklist.txt /etc/dnsmasq.d/

# Unbound
sed 's/^0\.0\.0\.0 /local-zone: "/' dynamic-blocklist.txt | \
  sed 's/$/" static/' | sudo tee /etc/unbound/blocklist.conf
```
