**DNS Blocklist Manager** — это специализированное программное обеспечение для ЭВМ БЭСМ-6, предназначенное для автоматизации формирования списков блокировки (blocklists). Версия 2.0 представляет собой оптимизированный рефакторинг алгоритмов обработки данных для эффективной работы в рамках ограничений архитектуры БЭСМ-6.



---

## 📋 Основные возможности

* **Эффективная хэш-таблица:** Использование алгоритма DJB2 (размер 10007) для обеспечения высокой скорости поиска и исключения дубликатов.
* **Гибкая фильтрация:**
    * **Whitelist:** Защита доверенных доменов.
    * **Blacklist:** Принудительная блокировка.
    * **Wildcard:** Использование масок для блокировки поддоменов.
* **Автоматизация:** Полный конвейер от парсинга `sources.txt` до генерации итогового `hosts.txt`.
* **Управление состоянием:** Поддержка `dry-run` (тестовый прогон) и автоматическое резервное копирование (`backup`).
* **Мониторинг:** Встроенная система вывода статистики обработки в реальном времени.

## 🚀 Начало работы

### Требования к файлам
Для работы программы в текущей директории должны присутствовать:
1. `config.ini` — настройки режимов работы.
2. `sources.txt` — список источников URL.
3. `whitelist.txt`, `blacklist.txt`, `wildcard.txt` — списки доменов для фильтрации.

### Запуск
Точка входа в программу — метка `START`. При выполнении программа последовательно проходит через этапы инициализации, загрузки конфигурации, обработки списков и генерации результата.
